Get Started

Simplicity First

The minimal and modern UI helps you focus on finding similar components without having to manually search and compare data for ages.

Performant

The effective scaling of requests using the serverless backend results in very fast response times, which also allows many users to be served effectively. This multi-user capability ensures that each user can store their smililarity search request and also make new requests independently.

Cost Effective

The requests can be easily scaled by the backend. If there is little to no traffic it can be scaled to zero, resulting in little to no cost. If there are a lot of requests it can be scaled up without any problems and the billing is very price efficient.
